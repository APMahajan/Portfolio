# Atul Prashant Mahajan — Developer Portfolio

A sleek, premium, developer-focused portfolio website designed for backend, DevOps, and full-stack SDE roles. Built with clean HTML5, custom vanilla CSS, and lightweight JavaScript featuring ambient glows, a noise overlay, and interactive terminal elements.

🖥️ **Live Demo:** [https://apmahajan.github.io/Portfolio/](https://apmahajan.github.io/Portfolio/)

🛒 **E-commerce Project Showcase:** [https://elite-e-commerce-system-design.onrender.com/](https://elite-e-commerce-system-design.onrender.com/)

---

## ✨ Features

- **Terminal Panel Interface**: Interactive MacOS/Linux-style terminal showing system specifications, core focus areas, and command-line inputs.
- **Typing Rotator**: Dynamic role-switching title rotator mimicking a command prompt cursor.
- **Ambient Neon Glows**: Dark mode interface accented by vibrant, custom-tailored neon gradients (`#CCFF00`) and radial glows.
- **Scroll Progress Thread**: Custom retro vertical line indicator tracking read progress.
- **Reveal on Scroll**: Seamless reveal animation using the `IntersectionObserver` API.
- **Fully Responsive**: Optimized for all devices from high-res desktop monitors to mobile screens.

---

## 🛠️ Tech Stack

- **Markup:** Semantic HTML5
- **Styling:** Vanilla CSS3 (Custom properties, HSL color tokens, glassmorphism, flexbox, and grid layouts)
- **Logic:** Vanilla JavaScript ES6
- **Typography:** Space Grotesk (display), Inter (body), JetBrains Mono (monospaced)

## 🚀 Running Locally

To preview and run this portfolio on your local machine:

### Using Node.js (Recommended)
Launch a local development server using `http-server`:
```bash
npx http-server -p 8000
```
Then open [http://localhost:8000](http://localhost:8000) in your browser.

### Using Python
Alternatively, start the server using Python's built-in module:
```bash
python -m http.server 8000
```
Then open [http://localhost:8000](http://localhost:8000) in your browser.

## 🌐 Free Hosting Guide

### Option 1: GitHub Pages
1. Create a public repository named `portfolio` on GitHub.
2. Push this project code:
   ```bash
   git init
   git add .
   git commit -m "Deploy portfolio website"
   git branch -M main
   git remote add origin https://github.com/<your-username>/portfolio.git
   git push -u origin main
   ```
3. Go to **Settings** > **Pages**. Select the `main` branch and `/root` directory, then click **Save**.

### Option 2: Netlify
1. Log in to [Netlify](https://www.netlify.com/).
2. Drag and drop this folder directly into the upload dropzone.
